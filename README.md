
ノベルゲームエンジン「LINKS」の最低限必要なタグを一括で登録できるようにしたものです。
なお、使用できるのはテキストファイルのみとなります。

以下、使い方

1:変換したいテキストファイルのところへ「LINKS_Tool.exe」をコピー。その後、クリックして実行。

2:「タグを登録したいファイルを指定してください：」と画面に出ますので、変換したいファイル名を入力してください。

3:「タグ登録後のファイル名を入力してください:」と画面に出た後、変換後のファイル名を入力してください。

4:先ほど指定したファイル名で「LINKS」のタグが登録されたテキストファイルが作成されます。
　あとは、LINKSのSTRフォルダ内の「LINKS.txt」にコピーすれば動作します。
