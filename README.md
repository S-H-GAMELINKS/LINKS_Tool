
ノベルゲームエンジン「LINKS」、「ティラノスクリプト」、「吉里吉里2」、「Nscripter2」の最低限必要なタグを一括で登録できるようにしたものです。
なお、使用できるのはテキストファイルのみとなります。

以下、使い方

1:変換したいテキストファイルのところへ「LINKS_Tool.exe」をコピー。その後、クリックして実行。

2:「使用する機能を選択してください」
　「1: LINKS用タグ登録, 2:ティラノスクリプト用タグ登録, 3:吉里吉里2用タグ登録, 4:Nscripter2用タグ登録」
　「使用する機能は？：」
　と表示されますので、使用したい機能の数字を入力してください。

3:「タグを登録したいファイルを指定してください：」と画面に出ますので、変換したいファイル名を入力してください。

4:「タグ登録後のファイル名を入力してください:」と画面に出た後、変換後のファイル名を入力してください。

5:先ほど指定したファイル名で各種エンジンタグが登録されたファイルが作成されます。
　あとは、各種エンジンの指定のフォルダにコピーすれば動作します。
